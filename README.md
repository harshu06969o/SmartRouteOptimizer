# Adaptive Route Optimizer using Graph Algorithms

This project implements a constraint-aware route optimization system using core graph algorithms such as BFS and Dijkstra.

The goal is to compute optimal paths between locations while handling real-world constraints like blocked routes and multiple starting points.
